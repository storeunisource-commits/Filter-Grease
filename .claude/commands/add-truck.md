# /add-truck - เพิ่มหรือแก้ไขรถใน Master List

เปิด Google Sheets tab `รถ_Master` แล้วเพิ่ม/แก้ไขข้อมูลรถ

คอลัมน์: truck_no | driver | status | type | active

สถานะที่ใช้ได้: ใช้งาน / ว่าง / จอดซ่อม / จอดเคลม / ลางาน
ประเภทรถ: รถน้ำ / รถทั่วไป / แมคโค

ถ้าต้องการ deactivate รถ ให้ set active = FALSE (จะไม่โชว์ใน form)
