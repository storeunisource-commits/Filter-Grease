# /deploy - Deploy to GitHub Pages

Deploy the Filter-Grease webapp to GitHub Pages.

Steps:
1. Verify api.js has the correct GAS_URL set (not placeholder)
2. Stage all changes with git add
3. Commit with a descriptive message
4. Push to main branch
5. GitHub Pages auto-deploys from the main branch root
6. Confirm at: https://storeunisource-commits.github.io/Filter-Grease/
